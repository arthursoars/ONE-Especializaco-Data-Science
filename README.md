# 🚀 Oracle Next Education (ONE) - Challenges de Data Science

Bem-vindo ao meu repositório de desafios do programa **ONE (Oracle Next Education)** em parceria com a **Alura**. Este espaço é destinado a documentar minha evolução na trilha de Ciência de Dados, aplicando conceitos de Python, manipulação de dados e visualização para resolver problemas de negócio reais.

## 📌 Sobre o Programa
O programa ONE é uma iniciativa de educação tecnológica com foco em empregabilidade. Atualmente, estou na fase inicial da especialização, desenvolvendo habilidades práticas em análise de dados.

## 📂 Projetos e Challenges

Abaixo, você encontrará o progresso dos desafios propostos durante a formação:

### 1. 📊 Análise de Desempenho Alura Store (Finalizado)
* **Descrição:** Análise de 4 lojas de e-commerce para decidir qual unidade deve ser vendida para levantamento de capital.
* **Ferramentas:** Python, Pandas e Matplotlib.
* **Principais Aprendizados:** Concatenagem de dados, uso de `groupby`, `unstack` e visualização de faturamento e frete.
* **Pasta do Projeto:** [/Challenge-01-Alura-Store](challenge1)
